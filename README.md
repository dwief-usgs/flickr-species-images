Accessing Images of a Species Through the Flickr API
===============================================================================================================

-----------
Contacts:
-----------
Daniel Wieferich (dwieferich@usgs.gov)


-----------
Purpose:
-----------
This effort explores the possibility of accessing images from Flickr of a species of interest.
